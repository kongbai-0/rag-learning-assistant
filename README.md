# 知识，自有答案

让课程资料真正成为知识。

上传教材、课件与讲义。

提问、总结、出题、复习。

所有回答都来自你的课程内容。

---

## 为学习而设计

上传 PDF 或 PPT。

系统自动构建课程知识库。

无须翻阅数百页教材。

直接提出问题。

获得基于课程资料的回答。

---

## 不只是问答

### 智能问答

从课程内容中检索答案。

提供引用来源。

减少大模型幻觉。

### 章节总结

快速掌握重点。

提炼核心概念。

聚焦考试内容。

### 自动出题

生成选择题、判断题与简答题。

附带答案与解析。

### 知识点解释

复杂概念。

用更容易理解的方式重新讲述。

### 学习追踪

记录学习过程。

发现薄弱知识点。

帮助持续复习。

---

## 工作方式

```text
课程资料
    ↓
构建知识库
    ↓
语义检索
    ↓
大语言模型
    ↓
生成回答
```

每一次回答。

都先检索。

再生成。

## 快速开始

安装依赖

```bash
pip install -r requirements.txt
```

配置环境变量

```env
DASHSCOPE_API_KEY=your_api_key
```

启动项目

```bash
python app_web.py
```
